# Navbar Blocks (16)

Navigation bars and menu components for site headers. Provide intuitive site-wide navigation.

## navbar1

![screenshot](../images/navbar/navbar1.png)

**Install**: `pnpm dlx shadcn add @shadcnblocks/navbar1`

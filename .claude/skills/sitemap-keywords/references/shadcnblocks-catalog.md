## Section Selection Tips

1. **Homepage** needs variety: Hero + trust (logos/testimonial) + features + social proof + CTA
2. **Service pages** focus on what you offer: Hero + services/feature + process + CTA
3. **About pages** build trust: Hero + about + team + timeline + CTA
4. **Contact pages** are conversion-focused: Contact form + FAQ (to handle objections)
5. **Pricing pages** need clarity: Hero + pricing + feature comparison + FAQ + CTA

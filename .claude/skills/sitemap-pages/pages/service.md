# Service (Individual) Template

## Global Sections
- Header (Navbar)
- Footer

## Core Sections (Required)
1. Hero - Specific service value prop
2. Feature - Key benefits/outcomes
3. Process - Step-by-step methodology
4. Pricing - Service-specific pricing
5. FAQ - Service-specific questions
6. CTA - Book consultation/get quote

## Optional Sections
- Testimonial - Service-specific reviews
- Casestudies - Relevant success stories
- Compare - Service tiers/options

## Conversion Goal
Primary: Consultation booking, quote request
Secondary: Contact inquiry, related services

## User Journey
From: Services overview, organic search
To: Contact, pricing, booking flow

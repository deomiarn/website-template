# Landing Page Template

## Global Sections
- Header (Navbar - minimal)
- Footer (minimal)

## Core Sections (Required)
1. Hero - Single focused value prop, CTA
2. Feature - Core benefits (3-5)
3. Stats - Social proof numbers
4. Testimonial - Customer validation
5. Pricing - Single plan or special offer
6. FAQ - Objection handling
7. CTA - Final conversion push

## Optional Sections
- Logos - Trust indicators
- Compare - vs. alternatives
- Banner - Urgency/scarcity element

## Conversion Goal
Primary: Single conversion action (signup/purchase/demo)
Secondary: Email capture for nurture

## User Journey
From: Paid ads, email campaigns, social media
To: Conversion action, thank you page

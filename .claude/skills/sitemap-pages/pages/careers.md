# Careers Template

## Global Sections
- Header (Navbar)
- Footer

## Core Sections (Required)
1. Hero - Company culture/why join
2. About - Values/mission/benefits
3. Careers - Open positions listings
4. Team - Current team/culture photos
5. CTA - Apply or submit general interest

## Optional Sections
- Stats - Company growth/employee metrics
- Testimonial - Employee testimonials
- Timeline - Company growth story
- FAQ - Hiring process questions

## Conversion Goal
Primary: Job application submission
Secondary: General interest/talent pool signup

## User Journey
From: About page, job boards, social media
To: Application form, contact, about

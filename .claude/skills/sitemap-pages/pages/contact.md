# Contact Template

## Global Sections
- Header (Navbar)
- Footer

## Core Sections (Required)
1. Hero - Contact purpose/channels overview
2. Contact - Multi-field form + map/info
3. FAQ - Common pre-contact questions
4. CTA - Alternative contact methods

## Optional Sections
- Stats - Response time/support metrics
- Team - Key contact people
- Testimonial - Support/service feedback

## Conversion Goal
Primary: Form submission, inquiry generation
Secondary: Phone call, email direct contact

## User Journey
From: Any page (global nav access)
To: Confirmation page, homepage, services

# Pricing Template

## Global Sections
- Header (Navbar)
- Footer

## Core Sections (Required)
1. Hero - Pricing philosophy/transparency
2. Pricing - Tiered plans/options
3. Compare - Feature comparison table
4. FAQ - Pricing-specific questions
5. Testimonial - Value validation
6. CTA - Start trial/get quote

## Optional Sections
- Stats - Customer savings/ROI metrics
- Feature - Plan highlights/best for
- Contact - Custom pricing inquiry

## Conversion Goal
Primary: Plan selection, trial start, purchase
Secondary: Contact for custom pricing

## User Journey
From: Homepage, services, product pages
To: Signup flow, contact, feature pages

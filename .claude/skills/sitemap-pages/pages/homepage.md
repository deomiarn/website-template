# Homepage Template

## Global Sections
- Header (Navbar)
- Footer

## Core Sections (Required)
1. Hero - Primary value prop, CTA above fold
2. Stats - Social proof through numbers
3. Feature - Core offering highlight
4. Feature - Secondary offering highlight
5. Testimonial - Customer validation
6. Pricing - Conversion driver
7. CTA - Primary action reinforcement
8. FAQ - Objection handling

## Optional Sections
- Logos - Trust indicators (clients/partners)
- Integration - Ecosystem compatibility
- Blog - Fresh content preview

## Conversion Goal
Primary: Drive main CTA (signup/demo/purchase)
Secondary: Newsletter capture, feature exploration

## User Journey
From: Organic search, direct traffic, referrals
To: Product pages, pricing, signup flow

# Portfolio Template

## Global Sections
- Header (Navbar)
- Footer

## Core Sections (Required)
1. Hero - Portfolio overview/specialization
2. Gallery - Project thumbnails grid
3. Casestudies - Featured work deep-dives
4. Testimonial - Client feedback
5. CTA - Hire/collaborate request

## Optional Sections
- Stats - Project metrics (completed, industries)
- Process - How we work
- Awards - Recognition/achievements
- Services - Related offerings

## Conversion Goal
Primary: Project inquiry, collaboration request
Secondary: Case study engagement, credibility building

## User Journey
From: Services, about, organic search
To: Individual case studies, contact, services

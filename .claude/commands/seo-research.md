# SEO Research

Conduct keyword research and competitive analysis.

## Input Required

```
Primary Keyword:
Location:
Business Type:
Known Competitors: (optional URLs)
```

## What This Does

Uses `seo-keyword-research` skill to:
- Discover competitors ranking for target keywords
- Find primary, secondary, and long-tail keywords
- Classify search intent per keyword
- Map keywords to specific pages
- Identify content gaps

## Output

- docs/seo-analysis.md

## When to Use

- At project start (via /init)
- When targeting new keywords
- When expanding to new markets

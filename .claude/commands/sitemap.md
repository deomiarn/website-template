# Sitemap

Create or update the website sitemap structure.

## Input Required

```
Business Type:
Business Name:
Location:
Primary Goal:
Target Audience:
Unique Selling Points:
Number of Pages:
```

## What This Does

Uses `sitemap-structure` skill to:
- Define all pages and routes
- Select shadcnblocks sections for each page
- Identify required legal pages (Impressum, Datenschutz)

## Output

- docs/sitemap.md

## When to Use

- At project start (via /init)
- When restructuring pages
- When adding new pages

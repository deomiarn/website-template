# Step 08: SEO Content Optimization

**Use the `seo-content-optimization` skill to complete this task.**

## INPUT

**Page:** (e.g., homepage, /leistungen, /kontakt)

---

## TASK

Optimize content for this page's target keywords from docs/seo-strategy.md.

### 1. Read Keyword Targets

Get from docs/seo-strategy.md:
- Primary keyword
- Secondary keywords
- Ranking target

### 2. Optimize Content

**H1 (Required):**
- Must contain primary keyword
- Only ONE H1 per page
- Compelling and natural

**Headings (H2/H3):**
- Include secondary keywords
- Question format for FAQ sections

**Body Text:**
- Keyword in first 100 words
- 1-2% keyword density
- Natural variations and synonyms

**Images:**
- Alt text with relevant keywords
- Descriptive, not stuffed

### 3. Update i18n Files

Apply optimized text to:
- messages/de.json
- messages/en.json

### 4. Meta Tags

Create for this page:
- Meta title (50-60 chars) with primary keyword
- Meta description (150-160 chars) with CTA

Verify keyword placement is natural and density is 1-2%.

Next: Continue with next page or run /final-release when all pages done

# SEO Content

Optimize page content for search engines.

## Input Required

```
Page Route: (e.g., /, /leistungen)
```

## What This Does

Uses `seo-content-optimization` skill to:

**Read:** docs/seo-analysis.md for page keywords

**Optimize H1:**
- Include primary keyword
- One H1 per page (Hero section)

**Optimize H2/H3:**
- Include secondary keywords
- Natural variations

**Optimize body text:**
- Primary keyword in first 100 words
- 1-2% keyword density
- Natural variations and synonyms
- Avoid keyword stuffing

**Optimize images:**
- Alt text with relevant keywords

**Create meta tags:**
- Meta title: 50-60 characters with primary keyword
- Meta description: 150-160 characters with CTA

## Output

- Updated i18n content
- Meta tags in page/layout

## When to Use

- After /page completes (manual step)
- When targeting new keywords
- When improving search rankings
